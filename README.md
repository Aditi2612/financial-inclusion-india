# Financial Inclusion in India — Banking Access Analysis

## Question
How does banking access differ between rural and urban India, and which districts lag most?

## Data
RBI Basic Statistical Return (BSR-1), via data.rbi.org.in
Covers: Year, State, District, Population Group (Rural/Urban), Bank Group, Occupation, credit amounts
